# Task Flow

Aplicativo de gerenciamento de tarefas desenvolvido com Ionic Framework, Angular e Firebase Firestore.

## Funcionalidades

- Criar tarefas
- Editar tarefas
- Concluir tarefas
- Reabrir tarefas
- Excluir tarefas
- Persistência de dados em nuvem
- Atualização em tempo real

## Tecnologias

- Ionic Framework
- Angular
- TypeScript
- Firebase Firestore
- HTML
- SCSS

## Estrutura

```text
src/
├── models
├── pages
├── services
└── environments
```

## Como executar

```bash
npm install
npm install firebase @angular/fire
ionic serve
```
