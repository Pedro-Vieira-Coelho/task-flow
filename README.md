# Task Flow

Task Flow is a task management application developed using Ionic Framework, Angular and Firebase Firestore.

## Features

* Create tasks
* Edit tasks
* Complete tasks
* Reopen completed tasks
* Delete tasks
* Real-time synchronization
* Persistent cloud storage

## Technologies

* Ionic Framework
* Angular
* TypeScript
* Firebase Firestore
* HTML
* SCSS

## Project Structure

```text
src/
├── app/
│   ├── models/
│   ├── pages/
│   ├── services/
│   └── app.routes.ts
├── environments/
└── assets/
```

## Installation

Install dependencies:

```bash
npm install
```

Install Firebase:

```bash
npm install firebase @angular/fire
```

Run the project:

```bash
ionic serve
```
